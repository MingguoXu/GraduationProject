# GraduationProject
毕业设计相关代码

1.HOG+SVM行人检测
基于python和opencv编写的代码，应用经典的HOG+SVM算法实现行人检测
目录说明：
 1. **Negative**：负样本数据集；
 2. **Positive**：正样本数据集；
 3. **TestData**：测试数据集；
 4. **pedestran_detect.py**：使用HOG+SVM进行训练+测试代码。
详细可参考博客：
https://blog.csdn.net/hongbin_xu/article/details/79845290

2.matlab

目录说明：
hog：HOG特征提取
Simulation：PID控制仿真，模糊推理系统，模糊PID控制仿真
